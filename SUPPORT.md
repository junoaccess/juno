# Support

Thank you for using Juno! This document provides information on how to get help and support.

## Getting Help

### Documentation

Before seeking help, please check our comprehensive documentation:

- [README](README.md) - Project overview and quick start guide
- [Contributing Guide](CONTRIBUTING.md) - Development setup and contribution guidelines
- [Documentation Site](#) - Full documentation (when available)

### GitHub Discussions

For questions, ideas, and general discussions:

- Visit our [GitHub Discussions](https://github.com/usejuno/juno/discussions)
- Search existing discussions to see if your question has been answered
- Start a new discussion if you cannot find an answer

### GitHub Issues

For bug reports and feature requests:

- Check [existing issues](https://github.com/usejuno/juno/issues) first
- Use our issue templates to provide necessary information
- Bug reports should include reproduction steps and environment details
- Feature requests should explain the use case and benefits

## Community Resources

### Email Support

For general inquiries: support@junoaccess.site

For security issues: security@junoaccess.site

For Code of Conduct violations: conduct@junoaccess.site

### Response Times

- GitHub Issues: We aim to respond within 2-3 business days
- GitHub Discussions: Community-driven, responses vary
- Security Issues: Acknowledged within 48 hours
- Email: Response within 3-5 business days

## What to Include in Your Request

When seeking support, please include:

- **Clear description** of your issue or question
- **Environment details**:
  - PHP version
  - Laravel version
  - Node version
  - Database type and version
  - Operating system
- **Steps to reproduce** (for bugs)
- **What you've tried** already
- **Relevant code** or error messages
- **Screenshots** if applicable

## What We Cannot Help With

While we want to be as helpful as possible, please note that we cannot provide:

- Custom development for specific projects
- Troubleshooting for significantly modified versions of Juno
- Support for unsupported versions
- Debugging of third-party packages or services

## Contributing

If you've found a bug or have a feature request, consider contributing directly:

- Read our [Contributing Guide](CONTRIBUTING.md)
- Submit a pull request with fixes or improvements
- Help others by answering questions in discussions

## Commercial Support

For organisations requiring dedicated support, consulting, or custom development:

- Email: enterprise@junoaccess.site

## Stay Updated

- Watch our [GitHub repository](https://github.com/usejuno/juno) for updates
- Check the [CHANGELOG](CHANGELOG.md) for recent changes
- Follow release notes for new features and breaking changes

Thank you for being part of the Juno community!
